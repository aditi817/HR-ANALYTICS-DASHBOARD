##  Power BI Dashboard Project

This project demonstrates an interactive Power BI dashboard created using Power BI Desktop.

###  Project Files
- `Dashboard.pbix` – Power BI source file
- `reports/Dashboard.pdf` – Exported Power BI report
- `screenshots/dashboard.png` – Dashboard preview image

###  Tools Used
- Power BI Desktop
- Microsoft Excel / CSV (for data source)

###  Objective
To analyze the dataset and present insights using interactive visuals such as KPIs, charts, and filters.

###  Key Insights
- Identified key performance indicators (KPIs)
- Analyzed trends and patterns in the data
- Created an easy-to-understand visual summary for decision-making
